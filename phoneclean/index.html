Privacy Policy

1. 접근 권한 및 처리하는 데이터

1. Permissions and Data Processed

본 앱은 사용자가 각 기능을 선택한 경우에 한해 iOS 권한 요청을 통해 연락처, 사진, 캘린더 데이터에 접근할 수 있습니다. 연락처 데이터(이름, 전화번호 등)는 연락처 정리 기능 제공을 위해, 사진 데이터는 사진 정리 기능 제공을 위해, 캘린더 데이터는 일정 정리 기능 제공을 위해 사용자 기기 내에서만 처리됩니다.

This app may access Contacts, Photos, and Calendar data only when the user chooses the relevant feature and grants permission through the iOS permission prompt. Contact data (such as names and phone numbers) is processed only on the user's device to provide contact cleaning features, photo data is processed only on the user's device to provide photo cleaning features, and calendar data is processed only on the user's device to provide calendar cleaning features.

2. 데이터 사용 목적

2. Purpose of Data Use

해당 데이터는 사용자가 선택한 항목을 검토, 정리 및 삭제할 수 있도록 돕기 위한 목적으로만 사용됩니다. 모든 삭제 또는 수정 작업은 사용자가 직접 시작하고 확인한 경우에만 수행됩니다.

This data is used solely to help the user review, organize, and delete items selected by the user. Any deletion or modification is performed only when initiated and confirmed by the user.

3. 데이터 저장, 전송 및 공유

3. Data Storage, Transmission, and Sharing

연락처, 사진, 캘린더 데이터는 개발자 서버로 수집, 업로드, 전송 또는 저장되지 않습니다. 또한 해당 데이터는 어떠한 제3자와도 공유, 판매 또는 마케팅 목적으로 사용되지 않습니다. 앱 운영에 필요한 최소한의 설정 정보 또는 임시 데이터만 사용자 기기 내부에 저장될 수 있습니다.

Contacts, Photos, and Calendar data is not collected, uploaded, transmitted to, or stored on the developer's servers. This data is not shared with, sold to, or used for marketing by any third parties. Only minimal app settings or temporary data necessary for app operation may be stored locally on the user's device.

4. 사용자 권한 및 통제

4. User Control

사용자는 언제든지 iOS 설정에서 본 앱의 연락처, 사진, 캘린더 접근 권한을 허용 또는 철회할 수 있습니다. 권한을 허용하지 않더라도 해당 권한이 필요한 기능을 제외한 앱의 다른 기능은 계속 사용할 수 있습니다.

Users can allow or revoke this app's access to Contacts, Photos, and Calendar at any time in iOS Settings. Even if permission is denied, users can continue to use other parts of the app except for features that require those permissions.

5. 데이터 보관 및 삭제

5. Data Retention and Deletion

본 앱은 연락처, 사진, 캘린더 데이터의 서버 측 사본을 보관하지 않습니다. 앱 내부에 저장된 최소한의 설정 정보 또는 임시 데이터는 사용자가 앱을 삭제하면 함께 삭제됩니다.

This app does not retain any server-side copies of Contacts, Photos, or Calendar data. Any minimal settings or temporary data stored within the app is removed when the app is deleted from the device.
